# PAGE-X

**PAGE-X** is a pagination tool for correctly formatting long images or scrolling screenshots into standard, printable A4/Letter page format.

[![CI](https://github.com/SeanVasey/PAGE-X/actions/workflows/ci.yml/badge.svg)](https://github.com/SeanVasey/PAGE-X/actions/workflows/ci.yml)

## Features

- Accepts long/scrolling screenshots as input
- Splits images into correctly sized A4/Letter pages
- Maintains readability and formatting across page breaks
- Outputs print-ready paginated documents
